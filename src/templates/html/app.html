app module¶

app.py start an API to populate a database.

app.create_jobs()[source]¶

POST method when no arguments are provided in the URL. This method is used to insert jobs in the database. raise: error if “priority” is not an integer or not between 0 and 2. task and status are also raising error if they are not respectivly in [TRAP, IL, TAP] and [success, pending, failure, retry, abort] return: https response code rtype: str, int

app.db_connection(database)[source]¶

Test connection with sqlite database raise: error connection return: connection to database rtype: sqlite3.Connection

app.delete_jobs(job_id)[source]¶

DELETE method for the API. It’s mainly used to delete a row in the database. param: URL id type: int return : https response code rtype : str, int

Index page for the API return: homepage rtype: html

app.query_jobs()[source]¶

GET method when no arguments are provided. This method return the content of the database. return: database rtype: json
