# Apipy 🐝

Apipy is a REST API in python.
It's more of a "proof of concept" than an actual project

## 🔧  Installation  

```sh 
git clone ... 
cd Apipy
pip install -r requirements.txt
```

*on your server :*    
`python3 src/api.py` 

*on your client :*  
`python3 src/client.py`
