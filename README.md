# Davinci 🐝

Davinci is a REST API in python built with FLASK.  
The server is running with gunicorn.  
The python version in `venv` is 3.8.10  

## 🔧  Installation  

```sh 
git clone ... 
cd Davinci 
source venv/bin/activate
pip install -r requirements.txt
```

*on your server :*    
`python3 main.py` 

*on your client :*   
`https://[your_ip]:5000`   
*or*   
`python3 src/client.py`
